# VarianceEstimation
This repository contains code and data for the experiments in paper: Variance Estimation and Selecting  Estimands for Causal Effect. We implemet two methods for variance estimation in order to distinguish which estimand to use for causal effect estimation, aiming to pick the estimand of minimum asymptotic variance. 
We use two approaches:
The **Plug-In estimator** and the ****boostrap** estimator.**
